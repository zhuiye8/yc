# components

大屏共享组件，被各大屏页面引用。

## Files

| File | Role | Responsibilities |
|---|---|---|
| header.jsx | UI | 大屏顶部标题栏 + 实时时钟（dayjs，每秒刷新） |
| bottom.jsx | UI | 大屏底部返回按钮（支持返回首页/返回上级） |
| map.jsx | UI | 中国地图 ECharts 组件，支持省份下钻到宜昌市，tooltip 自定义背景 |
