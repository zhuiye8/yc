# components (Index)

大屏首页五大模块子组件，每个组件展示对应领域的 KPI 概览与图表。

## Files

| File | Role | Responsibilities |
|---|---|---|
| industry.jsx | UI | 产业模块：链上企业总数 + 产业链 ECharts 环形图 |
| talent.jsx | UI | 人才模块：人才总数 + 分类统计 + ECharts 柱状图 |
| funds.jsx | UI | 资金模块：融资工具/投资机构/对接数 + ECharts 折线图 |
| innovation.jsx | UI | 创新模块：园区/项目/标准/知识产权 + ECharts 柱状图 |
| policy.jsx | UI | 政策模块：政策总数 + 分类统计卡片 |
