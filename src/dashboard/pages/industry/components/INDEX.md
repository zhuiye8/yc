# components (industry)

大屏产业布局页子组件。

## Files

| File | Role | Responsibilities |
|---|---|---|
| indicator.jsx | UI | 产业链核心指标（缺链数/强链数/产业链数/企业总数/人才总数） |
| enterprise.jsx | UI | 重点企业列表（企业名 + 所属行业） |
| distribution.jsx | UI | 企业类型分布 ECharts 柱状图（上市/专精特新/科技型中小/全量） |
| talent.jsx | UI | 产业链人才列表（人才名 + 职称 + 领域） |
