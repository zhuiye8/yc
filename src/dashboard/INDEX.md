# dashboard

大屏可视化子应用，使用 MUI Grid + 原生 ECharts（JSX），通过 Screen.tsx 嵌套路由接入主应用。
暗色全屏主题，1920×1080 固定布局。

## Subfolders

| Folder | Responsibilities |
|---|---|
| [components](components/INDEX.md) | 大屏共享组件（Header 时钟、Bottom 返回按钮、中国地图） |
| [pages/Index](pages/Index/INDEX.md) | 大屏首页（五大模块概览 + 地图） |
| [pages/industry](pages/industry/INDEX.md) | 大屏产业布局页 |
| [pages/talent](pages/talent/INDEX.md) | 大屏人才总览页 |
| [pages/funds](pages/funds/INDEX.md) | 大屏资金概览页（占位） |
| [pages/innovation](pages/innovation/INDEX.md) | 大屏创新资源页（占位） |
| [pages/policy](pages/policy/INDEX.md) | 大屏政策全景页（占位） |
